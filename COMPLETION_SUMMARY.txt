HOW TO USE
================================================================================

1. Streamlit Demo (No Credentials Needed)
   $ streamlit run streamlit_app.py
   → Opens UI with Ford analysis, trace metrics, guardrail results

2. MCP Server (Programmatic Access)
   $ python3 -m credit_research_agent.mcp.server
   → Exposes hybrid_retrieve and verify_numeric_claim tools

3. Run Tests
   $ python3 -m pytest tests/ -v
   → Verify all 39 tests pass

4. Full M3 Demo (Requires AWS Credentials)
   $ export AWS_REGION=us-east-1
   $ python3 scripts/run_m3_full_demo.py
   → Runs real Bedrock Claude, writes to runs/m3_full_demo/

See DELIVERY_GUIDE.md for detailed instructions.
